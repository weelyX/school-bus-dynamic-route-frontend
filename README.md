# School Bus Dynamic Route Frontend

An interactive school bus tracking web application frontend that connects school administration, students, and parents through a modern and responsive interface.

The project simulates a smart school transportation system with role-based screens, attendance tracking, route monitoring, and live map interaction using local mock data.

## Project Overview

This project was built as a frontend prototype for managing and monitoring school bus routes. It focuses on improving communication between school administrators, parents, and students by providing a clear interface for bus tracking and attendance management.

The system uses static frontend technologies and browser-based storage to simulate backend behavior without requiring a server.

## Features

- Responsive web interface
- School bus route tracking simulation
- Live map-style interaction
- Student and parent interface sections
- Administration dashboard concept
- Attendance management simulation
- Bilingual Arabic/English interface support
- LocalStorage and mock data simulation

## Tech Stack

- HTML
- CSS
- JavaScript
- LocalStorage
- Mock Data

## Project Files

- `index.html` — Main entry page for the school bus frontend application.
- `app.js` — Main JavaScript logic for interface behavior and data handling.
- `mock-data.js` — Mock data used to simulate backend information.
- `styles.css` — Styling file for layout, design, and responsiveness.

## How to Run

1. Clone or download the repository.
2. Open `index.html` directly in your browser.
3. Interact with the available pages and simulated data.

## Live Demo

The project can be deployed using GitHub Pages because it is built with static frontend files.
